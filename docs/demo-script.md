# Demo Script

1. Start in Workspace and point out that screen sharing is off by default.
2. Ask “Why is this component failing?” against the included demo code context.
3. Show the concise diagnosis, retrieved documents, confidence, and action suggestion.
4. Select the action, review the confirmation details, and cancel once to demonstrate control.
5. Approve the safe documentation action and show the activity ledger.
6. Open Knowledge to show the curated debugging, algorithm, browser, safety, and product packs.
7. Share a window, ask a new question, and explain that exactly one ephemeral frame is analyzed.
