# NeuraLens Product Guide

## Purpose

NeuraLens is a voice-first screen intelligence assistant. It combines user-controlled visual context, concise reasoning, curated local knowledge, and confirmation-first actions so people can move forward without repeatedly switching tools.

## Workspace modes

Coding mode focuses on errors, data flow, build output, and safe patches. Web task mode focuses on navigation, forms, comparison, and irreversible boundaries. Learning mode explains concepts and asks guiding questions. General mode gives focused next-step help.

## Visual context

The screen tile shows either a user-selected shared window or an explicit demo context. NeuraLens captures a compressed frame only when a question is sent or Analyze is selected. The user can stop sharing at any time.

## Answer contract

Answers identify the likely intent, state what can actually be observed, give no more than four next steps, surface confidence, cite local knowledge packs, and show action suggestions separately.

## Local mode

When a live reasoning provider is not configured or unavailable, NeuraLens uses hybrid keyword retrieval over its local Markdown library. Local mode remains useful for task guidance but does not claim detailed vision understanding.
